    Introduction

    pyUblox is a set of Python scripts that can do the following

    • Send and receive (pretty much) all binary uBlox messages
    • Solve the GPS equations from raw data (including decoding the Ephemeris, calculating sat positions, calculating basic corrections etc)
    • Generate RTCMv2 differential GPS corrections from a static uBlox reference station (raw data-capable only)
    • Generate RTCMv2 differential GPS corrections from an NTRIP/RTCMv3 source
    • Send the RTCMv2 corrections directly to serial port (typically another uBlox device)
    • Send the RTCMv2 corrections over UDP (typically to a MAVProxy DGPS module instance to be forwarded to a remote GPS)

    Scripts

    This is all accomplished using a small number of top-level scripts.

    All scripts that take input from a uBlox receiver can also have a binary uBlox file specified as input in order to replay data.

    dgps_test.py

    Generate DGPS corrections from a local receiver, send to local receiver.

    --port1 Reference receiver

    --port2 Receiver to correct

    --port3 Uncorrected control receiver (optional)

    --reference Location of reference receiver, LLH, comma-separated.

    dgps_ntriperrs.py

    Generate DGPS corrections from an NTRIP source, send to local receiver.

    --ntrip-* NTRIP server, mount and credential information (see -h output for details)

    --port2 Receiver to correct

    --port3 Uncorrected control receiver (optional)

    local_to_udp.py

    Generate DGPS corrections from a local receiver, send over UDP

    ntrip_to_udp.py

    Generate DGPS corrections from NTRIP, send over UDP

    ublox_*.py

    ublox_capture.py: Save ubx data to a file for later processing

    ublox_show.py: Print decoded uBlox messages to the console

    ublox_plot.py: Draw the receiver's location over time (in 2D)
